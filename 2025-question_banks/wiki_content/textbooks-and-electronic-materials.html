Prerequisite Knowledge

At this point in your path you should have experience with HTML, CSS, and SQL databases. If you are not yet comfortable with these technologies, please take some time to review the following resources. You are responsible for making sure you are comfortable with these basics.

Intro to HTML: https://developer.mozilla.org/en-US/docs/Learn/HTML/Introduction_to_HTML (Links to an external site.)

Intro to CSS: https://developer.mozilla.org/en-US/docs/Learn/CSS (Links to an external site.)

SQL practice: https://sqlzoo.net/wiki/SQL_Tutorial (Links to an external site.)

If you would like to get a head start on our course materials, you should review Git, HTTP, and JavaScript.

Using Git: https://github.com/jlord/git-it-electron (Links to an external site.)

HTTP: https://developer.mozilla.org/en-US/docs/Web/HTTP (Links to an external site.)

JavaScript: https://github.com/workshopper/javascripting (Links to an external site.) <-- Completing this tutorial is an assignment in week 2
