Client-Side Scripting

Lesson Objective:  At the end of this lesson, you will be able to:

  • Compare popular client-side and server-side programming languages, including JavaScript, Java, PHP, Python, .Net, C, C++, Visual Basic, C#.
  • Define Common Gateway Interface (CGI) methods, including .Net, Django, Python, JavaServer Pages (JSP), Server-Side JavaScript (SSJS), Active Server Pages (ASP), PHP Hypertext Preprocessor (PHP), Ajax.
  • Identify ways to use additional technologies to provide custom features to an end user (e.g., using JavaScript to detect Web browser type, using cookies).


The most common client-side scripting language is JavaScript.  There are others that exist, including VBScript (only supported by IE), and Microsoft's JScript (an extended version of JavaScript).  This is one battle that Microsoft has not won.  Web developers have adopted JavaScript to be the common language to work on the Client.

There are disadvantages, the end-user can disable JavaScript.  Browsers do not support all functions of JavaScript equally, so aim for the lowest common denominator when working with JavaScript. 

JavaScript

JavaScript is the primary client-side scripting language and allows the developer to add interest, interaction, form validation, dynamic and asynchronous changes to Web pages.   JavaScript can be added anywhere within your Web pages; it can be loaded from a separate Web document with a .js extension, it can even be included within actual HTML elements.  Example of JavaScript usage within HTML:

<!DOCTYPE html>
<html lang="en">
<head>
     <title>A Good Page Title Here</title>
     <meta charset="utf-8"/>
     <meta name="author" content="your name">
     <meta name="date" content="date created">
     <meta name="robots" content="NOINDEX, NOFOLLOW" />

     <!-- adding an external script -->
     <script src="myscripts.js" type="text/javascript"></script>

</head>
<body>
<!-- Embedding a script within a page -->
<script type="text/javascript">
<!--
   document.write("Hello World");
//-->
</script>

<!-- Script within an HTML element -->
<input type="button" onclick="alert('Thanks for clicking on me!');" value="Click here" />


</body>
</html>
