Document Object Model

Lesson Objective:  At the end of this lesson, you will be able to:

  • Define the Document Object Model (DOM) and its relationship to Dynamic HTML (DHTML).

 

When a page is loaded into the browser, there browser takes an inventory of all items that make up the page.  The inventory can elements, multimedia, forms, links; basically everything that makes up the page. 

The inventory becomes your playground, that you can change anytime that you want through the use of the DOM (document object model).  The DOM can also be called the JavaScript Hierarchy. 

 

Using the DOM (just a little fun)

Although your text does not show you any examples, I thought it might be fun for you to play with at least one simple example.  Scroll over the code below, save it as DOM.html and display it in your browser.

function DOM()
purpose:      Get input from the user and
                 add the information to a specific div
author:         Marti Baker
date:             12/19/2013
parameters:    none
return value:  true, let the calling position know that function
                    has been completed

function DOM(){

    var strUser = prompt("What is your first name?");
    
    // dynamically write the output to the window
    document.getElementById("output").innerHTML = "Thank you " + strUser + " for visiting our page. ";
    
    // hid the input button
    document.getElementById("input").style.visibility = "hidden";
    
    return true;
}

//]]>
-->
</script>
