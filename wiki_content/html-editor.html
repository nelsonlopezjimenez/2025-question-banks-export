HTML Editor

There are many freeware HTML editors available for use during your studies within this program.  I am a big proponent of freeware.  I've been a starving student, I know what it means to find it difficult to pay bills.  So when every possible, I find the freeware options for students.

 

Lesson Objective:  install and work with the essential tools / software used by web developers / designers: html editors, css editor, simple graphics editor or creators, and ftp software

 

Many students entering this program have experience writing HTML and using an HTML editor.  I encourage you to use the HTML editor that you are accustom to using as long as it has a code-based screen.  During these next modules, 18 through 26, you will be required to do most of your work from the code side, not the WYSIWYG side or design screen side.  When choosing an editor, I have some suggestions:

  1. Make sure the editor has a strong community, including a forum where you can post questions about the use of the editor
  2. Make sure the editor has good tutorials about how to get started and how to accomplish specific tasks
  • Suggested editors include:
    • Kompozer for Windows, Mac, and Linux OS.
    • Dreamweaver. By far the most commonly used editor in Web development.  However, it is expensive. Available for both Mac and Windows.
    • NetBeans IDE - Quickly and easily develop desktop, mobile and web applications with Java, HTML5, PHP, C/C++ and more.
    • Microsoft Expression Web 4. Free download works with Windows systems.
    • Notepad++ for Windows
      • A good video about downloading and installing a Notepad++ editor by Bucky Roberts of thenewboston.org (you will see a lot of Bucky in the next few months)XHTML and CSS Tutorial - 1 - Downloading a Text Editor
If none of these editors are to your liking, don't be afraid to do some more searching there are new editors put out there every day.  I would be sure that you do a virus scan on the downloaded software before you install these on to your computer.
By the way, my own personal choice for an editor, isn't available anymore. It's an oldie but a goodie, called Homesite.  Today, I'm working a bit more with Netbeans, another freebee.
